# Generator

Generated code depends on:

- `react`
- `react-native-navigation`
- `react-test-renderer`
- `jest`

## Install

Install [yeoman](https://yeoman.io/):

```
npm install -g yo
```

Link local project as npm package:

```
npm link
```

## Usage

```
yo beda
yo beda:app # Application
yo beda:component # React Native Container component
```
